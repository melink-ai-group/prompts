# Games Index

Source of truth for interactive game prompt inventory.

Browse prompts: [README.md](README.md) · Repo home: [../README.md](../README.md)

## Naming

- Pattern: `game__<name>.md`
- Style: lowercase, kebab-case, no spaces
- One game per file

## How to use

| Step | What to do |
|------|------------|
| 1 | Open a game `.md` file and copy all (`Cmd/Ctrl+A`) |
| 2 | Paste as your **first message** in a **new** chat |
| 3 | Send it — the host runs setup and starts the game |

Same quick-start flow as [personas](../personas/HOW%20TO%20GUIDE.md). Start a new chat to reset.
