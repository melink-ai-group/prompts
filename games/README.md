# Interactive AI Game Prompts

Copy-paste **interactive game prompts** for ChatGPT, Claude, Cursor, and Gemini — live game hosts that run entirely in chat. Not personas: these are rule sets, credits, and show framing.

Full inventory: [INDEX.md](INDEX.md) · Persona prompts: [personas/README.md](../personas/README.md)

---

## Quick start

| Step | What to do |
|------|------------|
| 1 | Open a game `.md` file and copy all (`Cmd/Ctrl+A`) |
| 2 | Paste as your **first message** in a **new** chat |
| 3 | Send it — the host runs setup and starts the game |

Start a new chat to play again.

## Games

### Kemp Knows Best

A live high-stakes guessing game. The host (Phillip Kemp) hides an answer in a chosen category — movies, TV, games, music, history, and more. You ask yes/no questions and spend credits until you guess or run out.

Features: category sharpening, difficulty levels (Easy through Kemp Knows Best), credit system, compound-question rules, live game-show tone.

**Prompt:** [game__kemp-knows-best.md](game__kemp-knows-best.md)

---

## What game prompts are

Game prompts are **interactive hosts** — they run setup, state, and rules in conversation. Persona prompts load a voice; game prompts load a **session**.

Keywords: AI guessing game, ChatGPT game prompt, interactive AI game, copy paste game host, Kemp Knows Best.
